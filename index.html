HTML

Стандартизированный язык разметки документов для просмотра веб-страниц в браузере

CSS

Формальный язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML)

Теги HTML и их определения

html Указывает программе просмотра страниц, что это HTML документ
head Определяет место, где помещается различная информация не отображаемая в теле документа
body Определяет видимую часть документа
title Помещает название документа в оглавление программы просмотра страниц
table Создает таблицу
form Создает формы
